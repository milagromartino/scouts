## 🚀 Instalación Local

```bash
# 1. Clonar o descargar el proyecto
cd scouts

# 2. Instalar dependencias (opcional, solo si quieres compilar Tailwind)
npm install

# 3. Para desarrollo con watch de cambios:
npm run dev

# 4. Para hacer build final:
npm run build
```

## 🌐 Deploy en Netlify

Este proyecto está completamente configurado para Netlify:

### Opción 1: Conectar con Git
1. Sube tu código a GitHub
2. Entra a [netlify.com](https://netlify.com)
3. Clic en "New site from Git"
4. Selecciona tu repositorio
5. Netlify hará el build automáticamente

### Opción 2: Drag & Drop
1. Descarga el proyecto
2. Abre [netlify.com/drop](https://netlify.com/drop)
3. Arrastra la carpeta del proyecto
4. ¡Listo! Tu sitio estará online

### Configuración Automática
Netlify usará estos archivos automáticamente:
- `netlify.toml` - Instrucciones de build
- `package.json` - Dependencias
- `tailwind.config.js` - Configuración de Tailwind
